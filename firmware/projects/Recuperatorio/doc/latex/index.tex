\chapter{Sistema de pesaje para camiones}
\hypertarget{index}{}\label{index}\index{Sistema de pesaje para camiones@{Sistema de pesaje para camiones}}
\hypertarget{index_genDesc}{}\doxysection{\texorpdfstring{General Description}{General Description}}\label{index_genDesc}
Este codigo es el encargado de manejar un sistema de pesaje, para esto mide la velocidad y distancia del vehiculo con la ayuda del sensor de distancia HC\+\_\+\+SR04, una vez que el vehiculo se encuentra detenido, realiza la medicion del peso como un promedio de 100 valores (50 para cada galga) y envia por UART el peso promedio y la velocidad maxima, a su vez cuenta con la funcionalidad de abrir una barrera con la tecla \textquotesingle{}o\textquotesingle{}, o cerrarla con la tecla \textquotesingle{}c\textquotesingle{}\hypertarget{index_hardConn}{}\doxysection{\texorpdfstring{Hardware Connection}{Hardware Connection}}\label{index_hardConn}
